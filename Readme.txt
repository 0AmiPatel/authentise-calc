How to run code document
Install nodeJS on your machine.
Install VS code
Open project in VScode
Complete github setup in VScode
Open terminal and Run “npm init” in terminal
Run npm install –save –dev cypress@10.7.0 to install Cypress version 10+
To open test runner: in open console in vscode and run: npx cypress open
Select E2E testing to run the code.
Select browser chrome (V111)> start E2E testing in chrome
Select Homepage.cy.js in Test Runner to run the code
